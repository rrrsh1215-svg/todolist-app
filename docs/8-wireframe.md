# TodoListApp 와이어프레임

## 1. 화면 목록

| ID | 화면 | 주요 기능 |
| --- | --- | --- |
| WF-001 | 로그인 | 이메일/비밀번호 로그인 |
| WF-002 | 회원가입 | 이메일/비밀번호/이름 입력 |
| WF-003 | 할일 목록 | 목록 조회, 필터링, 편집/삭제 진입 |
| WF-004 | 할일 등록/수정 | 할일 정보 입력, 상태 변경 |
| WF-005 | 내 정보 | 이름, 다크모드, 언어 저장, 로그아웃, 회원 탈퇴 |

## 2. WF-001 로그인

+--------------------------------+

+--------------------------------+
```

+--------------------------------+
```

## 4. WF-003 할일 목록

```text
+--------------------------------------------------------------------------------+
| TodoListApp                                                   [내 정보]          |
+--------------------------------------------------------------------------------+
| 할 일 목록                                                   [새 할 일]          |
+--------------------------------------------------------------------------------+
| 카테고리 [전체 v]  시작일 [yyyy-mm-dd]  종료일 [yyyy-mm-dd]  상태 [전체 v] [초기화] |
+--------------------------------------------------------------------------------+
| 주간 보고서 작성                         업무  2026-05-15  등록    [편집] [삭제] |
| 금요일 회의 전까지 초안 작성                                                   |
+--------------------------------------------------------------------------------+
| React Query 정리                         학습  2026-05-13  완료    [편집] [삭제] |
+--------------------------------------------------------------------------------+
```

필수 요소:

- 본인 소유 할일 목록
- 카테고리 필터
- 종료예정일 시작/종료 필터
- 상태 필터: 전체, 등록, 진행중, 완료, 취소
- 필터 초기화
- 새 할일 등록 진입
- 할일 편집, 삭제
- 필터 결과 없음 상태

목록 화면에서는 상태 변경 체크박스를 제공하지 않는다. 상태 변경은 편집 화면에서 수행한다.

## 5. WF-004 할일 등록/수정

```text
+------------------------------------------------------+
| 할 일 등록 / 수정                                    |
+------------------------------------------------------+
| 할 일명 *                                            |
| [                                                  ] |
| 설명                                                 |
| [                                                  ] |
| [                                                  ] |
| 종료예정일                                           |
| [YYYY-MM-DD]                                        |
| 카테고리 *                                           |
| [일반 v]                                             |
| 상태 (수정 화면에서 표시)                            |
| [등록 / 진행중 / 완료 / 취소 v]                      |
|                                                      |
|                              [취소] [저장]           |
+------------------------------------------------------+
| 카테고리 추가                                        |
| 새 카테고리명 [                 ] [추가]             |
+------------------------------------------------------+
```

필수 요소:

- 할일명 필수 입력
- 설명 입력
- 종료예정일 입력
- 기본/사용자 추가 카테고리 선택
- 수정 화면에서 상태 선택
- 카테고리 생성 UI
- 저장/취소

## 6. WF-005 내 정보

```text
+------------------------------------------------------+
| 내 정보                                      [할 일 목록] |
+------------------------------------------------------+
| 이메일                                               |
| [user@example.com]                                   |
| 이름 또는 닉네임                                     |
| [태수호]                                             |
| 언어                                                 |
| [한국어 / English / 日本語 v]                         |
| [x] 다크모드 사용                                    |
|                                                      |
|                              [취소] [저장]           |
+------------------------------------------------------+
| 회원 탈퇴                                            |
| [로그아웃] [회원 탈퇴]                               |
+------------------------------------------------------+
```

필수 요소:

- 이메일 읽기 전용 표시
- 이름 또는 닉네임 수정
- 표시 언어 선택
- 다크모드 체크박스는 `다크모드 사용` 텍스트 왼쪽에 배치
- 로그아웃
- 회원 탈퇴와 삭제 대상 안내

## 7. 다국어 표시 기준

기본 카테고리 표시:

| 원본 | 한국어 | 영어 | 일본어 |
| --- | --- | --- | --- |
| 일반 | 일반 | General | 一般 |
| 업무 | 업무 | Work | 業務 |
| 개인 | 개인 | Personal | 個人 |
| 학습 | 학습 | Study | 学習 |

상태 표시:

| API 값 | 한국어 | 영어 | 일본어 |
| --- | --- | --- | --- |
| `registered` | 등록 | Registered | 登録 |
| `in_progress` | 진행중 | In progress | 進行中 |
| `completed` | 완료 | Completed | 完了 |
| `canceled` | 취소 | Canceled | キャンセル |
