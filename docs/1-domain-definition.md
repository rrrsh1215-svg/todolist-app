# TodoListApp 도메인 정의서

## 1. 도메인 개요

TodoListApp은 인증된 개인 사용자가 자신의 할일을 데이터베이스에 저장하고, 카테고리와 일정 기준으로 체계적으로 관리하는 개인용 할일 관리 애플리케이션이다.

이 서비스는 사용자별 데이터 분리, 개인 할일 등록, 카테고리 기반 분류, 조건별 목록 조회를 핵심 도메인으로 한다.

## 2. 해결하려는 문제

- 개인 사용자가 본인 계정 기준으로 할일을 안전하게 저장하고 관리하기 어렵다.
- 인증된 사용자 단위로 할일 데이터가 분리되어 저장되는 구조가 필요하다.
- 할일을 카테고리별로 등록하고 조회할 수 있는 관리 기준이 필요하다.

## 3. 주요 행위자

### 비회원 사용자

- 회원가입을 통해 서비스 계정을 생성한다.
- 로그인 후 인증 사용자로 전환된다.

### 인증 사용자

- 본인의 개인 정보를 수정한다.
- 본인의 계정을 탈퇴할 수 있다.
- 본인의 할일을 등록, 조회, 수정, 삭제한다.
- 기본 카테고리와 직접 추가한 카테고리를 사용해 할일을 분류한다.
- 카테고리, 종료예정일 기간, 완료 여부 기준으로 할일 목록을 필터링한다.

### 시스템

- 사용자 인증 상태를 확인한다.
- 사용자별 데이터 접근 범위를 제한한다.
- 기본 카테고리를 제공한다.
- 조회 조건에 맞는 사용자 소유 할일만 반환한다.
- 회원 탈퇴 시 사용자 관련 데이터를 즉시 삭제한다.

## 4. 핵심 도메인 개념

### 사용자

서비스를 이용하는 개인 주체이다. 사용자는 인증 정보를 통해 식별되며, 본인의 개인 정보와 할일 데이터만 관리할 수 있다. 사용자는 계정 탈퇴를 통해 본인 계정과 관련 데이터를 삭제할 수 있다.

### 인증

사용자가 본인 계정으로 서비스를 이용하기 위한 절차이다. 할일, 카테고리, 개인 정보 관리는 인증된 사용자에게만 허용된다.

### 할일

사용자가 수행해야 할 작업 단위이다. 할일은 특정 사용자에게 귀속되며, 다음 정보를 가진다.

- 할일명
- 설명
- 종료예정일
- 카테고리
- 완료 여부

### 카테고리

할일을 분류하기 위한 기준이다. 카테고리는 기본 카테고리와 사용자 추가 카테고리로 구분된다.

### 기본 카테고리

시스템이 모든 사용자에게 공통으로 제공하는 카테고리이다. 사용자는 별도 생성 없이 기본 카테고리를 할일에 지정할 수 있다. MVP 기본 카테고리는 일반, 업무, 개인, 학습이다.

### 사용자 추가 카테고리

인증 사용자가 직접 추가한 개인 카테고리이다. 해당 카테고리는 생성한 사용자에게만 귀속되며, 다른 사용자는 사용할 수 없다.

## 5. 주요 업무 규칙

- 사용자는 회원가입 및 인증 후 앱을 사용할 수 있다.
- 인증 사용자는 본인의 개인 정보만 수정할 수 있다.
- 인증 사용자가 회원 탈퇴하면 사용자 계정, 본인 소유 할일, 사용자 추가 카테고리는 즉시 삭제된다.
- 인증 사용자는 본인 소유의 할일만 등록, 조회, 수정, 삭제할 수 있다.
- 사용자는 다른 사용자의 할일, 개인 정보, 사용자 추가 카테고리에 접근할 수 없다.
- 할일 등록 시 할일명과 카테고리는 필수이다.
- 할일은 하나의 카테고리에 속한다.
- 할일의 카테고리는 기본 카테고리 또는 본인이 추가한 카테고리 중에서 선택한다.
- 할일의 완료 여부는 완료 또는 미완료 상태로 관리한다.
- 할일 목록은 카테고리, 종료예정일 기간, 완료 여부 조건으로 필터링할 수 있다.
- 여러 필터 조건은 함께 적용할 수 있다.

## 6. 주요 유스케이스

### 회원가입 및 로그인

사용자는 계정을 생성하고 로그인한다. 시스템은 인증에 성공한 사용자에게 개인 데이터 접근 권한을 부여한다.

### 개인 정보 수정

인증 사용자는 본인의 개인 정보를 수정한다. 시스템은 수정 대상 사용자가 요청 사용자 본인인지 확인한다.

### 회원 탈퇴

인증 사용자는 본인의 계정을 탈퇴한다. 시스템은 요청 사용자가 본인인지 확인하고, 사용자 계정과 본인 소유 할일 및 사용자 추가 카테고리를 즉시 삭제한다.

### 할일 등록

인증 사용자는 할일명, 설명, 종료예정일, 카테고리를 입력해 할일을 등록한다. 시스템은 등록된 할일을 요청 사용자에게 귀속시킨다.

### 카테고리 추가

인증 사용자는 기본 카테고리 외에 본인이 사용할 카테고리를 추가한다. 시스템은 추가된 카테고리를 해당 사용자에게 귀속시킨다.

### 할일 목록 조회

인증 사용자는 본인의 할일 목록을 조회한다. 시스템은 카테고리, 종료예정일 기간, 완료 여부 필터를 적용해 조건에 맞는 할일만 반환한다.

## 7. 범위 제외

- 팀 또는 조직 단위의 공동 할일 관리
- 할일 공유 및 담당자 배정
- 반복 일정
- 알림 및 리마인더
- 파일 첨부
- 댓글 및 활동 이력
- 외부 캘린더 연동
- 관리자 기능
